# 🚀 Employee Task Management System

A simple and interactive **Employee Task Management Web App** built using **React.js**.
This project allows admins to assign tasks and employees to track their work progress.

---

## ✨ Features

* 🔐 Admin & Employee Login
* 📋 Task Assignment & Tracking
* 📊 Task Status (Active, New, Completed, Failed)
* 💾 Data stored using LocalStorage
* ⚡ Fast and responsive UI (Tailwind CSS)

## 📌 How It Works

* Data is initialized using `localStorage`
* On first load, default employees & admin are stored
* Login system checks credentials from stored data
* Tasks are managed per employee
